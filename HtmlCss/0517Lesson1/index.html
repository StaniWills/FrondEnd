Einstig Visual Studio-Code in Aktion

Visual Studio Code ist ein leichter, aber leistungsstarker Quellcode-Editor,

der auf Ihrem Desktop ausgeführt wird und für Windows, macOS und Linux verfügbar ist.

Es bietet integrierte Unterstützung für JavaScript, TypeScript und Node.js und verfügt über ein reichhaltiges

Ökosystem von Erweiterungen für andere Sprachen (wie C++, C#, Java, Python, PHP, Go) und Laufzeiten (wie .NET und Unity). .
